Prestige - Portfolio WordPress Theme

Prestige is a minimal and colorful portfolio WordPress theme based on menu tabs in different colors. Tabs are presenting pages content. Each page is one Prestige tab.

  • jQuery Powered
  • Unlimited number of tabs
  • Tab background colors defined from colorpicker
  • 7 different page changing animations available
    • expand
    • scroll
    • directscroll
    • fade
    • crossfade
    • cover
    • uncover
  • touch event support for changing pages
  • 20 tab icons
  • 10 predefined social icons
  • Fully customizable
  • Blog template
  • Build-in portfolio with Lightbox
  • Build-in resume implemented with shortcode
  • Build-in references implemented with shortcode
  • Build-in Google Map shortcode
  • Working Contact Form with configuration page under Settings submenu
  • Latest Tweets
  • Font Face Fonts
  • Valid XHTML Code
  • Documentation Included

On this video you can see how to activate theme and import dummy-content to it (from included - prestige-content.xml file).

To see how to add new tab/page please watch this video.

CSS Styles

The project uses the following cascade of template styles (in order):

  • style/jquery.jScrollPane.css - default styles for scrollbars,
  • style/jquery.qtip.css - default styles for tooltips,
  • style/jquery.nivo-slider.css - default styles for slider,
  • style/fancybox/jquery.fancybox.css - default styles for fancybox,
  • style.css - base styles for whole project.

Theme Structure

Project is divided by four main files: index.php, header.php, footer.php and functions.php


index.php

It's main project file which calls other. It contains get_header and get_footer calls. There is also call of the prestige_get function, which generate the menu tabs.


header.php

The file contains head section of the html tree structure and also head section of the page such as header logo area and Twitter notes (configurable via Theme Options page). You can find here also wp_head call.


footer.php

The file contains bottom part of the page, including copyright area and footer menu area (configurable via Menus page). You can find here also wp_footer call.


funstions.php

The file contains definition/includes of the functions for:

  • contact form, map,
  • portfolio, resume, references shortcodes,
  • footer menu,
  • blog sidebar,
  • ajax calls,
  • load javascript and css files,
  • base configuration,
  • prestige menu tab generation,
  • custom meta box and options pages.

Portfolio

Prestige have custom post type for portfolio. You can manage portfolio items in your admin area under Portfolio submenu.
Basically to put portfolio on your page, just use below shortcode: 

[prestige_portfolio]

There are four types of portfolio item available: image, video, iframe, , external url. To use video item, you need to set its url in Video URL (optional) textfield, analogously for iframe and external url.

You can also display portfolio items from certain caregories. Examples: 

[prestige_portfolio category='image']
[prestige_portfolio category='video']
[prestige_portfolio category='image,video']

Resume

Prestige have custom post type for resume. You can manage resume items in your admin area under Resume submenu.
Basically to put resume on your page, just use below shortcode: 

[prestige_resume]

You can also display resume items from certain caregories and with some header text (default is 'Work Experience'). Examples: 

[prestige_resume header='Work Experience' category='work_experience']
[prestige_resume header='Education' category='education']

References

Prestige have custom post type for references. You can manage references items in your admin area under References submenu.
Basically to put resume on your page, just use below shortcode: 

[prestige_references]

You can also display references items from certain caregories and with some header text (default is 'Clients references'). Example: 

[prestige_resume header='Clients references' category='websites']

You can also use simple display style: 

[prestige_resume header='Clients references' style='simple']

Contact form

This project includes a contact form ready for use.
All you need is to configure some constants located under Settings submenu (Prestige Contact Form Options position). Then you can use in page content below shortcode: 

[prestige_contact_form]

Google Map

Prestige includes also Google Map shortcode.
It can be added to the page with [prestige_map] shortcode. Example: 

[prestige_map width="298" height="175" lat="29.760193" lng="-95.36939" marker_lat="29.760193" marker_lng="-95.36939"]

Available shortcode parameters:

Parameter Default value Description
id map Specifies the identifier of the map. Should be unique for each map, if they are displayed in the same page.
width 298 Specifies the width of the map in pixels.
height 150 Specifies the height of the map in pixels.
lat 29.760193 Specifies the latitude coordinate of the map center point.
lng -95.36939 Specifies the longitude coordinate of the map center point.
marker_lat 29.760193 Specifies the latitude coordinate of the map marker.
marker_lng -95.36939 Specifies the longitude coordinate of the map marker.
zoom 10 Specifies the zoom value for the map.

Social icons

There is shortcode which help to create social icon or social icons list with links to social profiles.
To add for example Twitter icon, please use: 

[prestige_social_icon_list]
	[prestige_social_icon_list_item type='twitter' url='http://twitter.com/QuanticaLabs' tooltip='Twitter' target='_blank']
[/prestige_social_icon_list]

You can add more icons on one list: 

[prestige_social_icon_list]
	[prestige_social_icon_list_item type='twitter' url='http://twitter.com/QuanticaLabs' tooltip='Twitter' target='_blank']
	[prestige_social_icon_list_item type='facebook' url='#' tooltip='Facebook']
[/prestige_social_icon_list]

You can display social icons in other way - with description next to them. Example: 

[prestige_social_icon_list header=""]
	[prestige_social_icon_list_item type='twitter' url='http://twitter.com/QuanticaLabs' target='_blank']Twitter Profile[/prestige_social_icon_list_item]
	[prestige_social_icon_list_item type='facebook' url='http://facebook.com/ruth.howell' target='_blank']Facebook Profile[/prestige_social_icon_list_item]
[/prestige_social_icon_list]

Skill list

There is shortcode which help to create skill list.
Example: 

[skill_list header='Graphics and media skills']
	[skill_list_item level='9']Photoshop[/skill_list_item]
	[skill_list_item level='5']Illustrator[/skill_list_item]
	[skill_list_item level='6']InDesign[/skill_list_item]
	[skill_list_item level='4']Flash[/skill_list_item]
[/skill_list]

For skill_list shortcode you can define header text using header parameter.

For skill_list_item you can define level parameter which will sets the level of skill under skill item (can be between 0 and 10) and style parameter which can be set to simple.

Languages

Theme supports translations. To create your language translation, please download this usefull tool Poedit. Then find in languages directory default.po file and open it in Poedit. Translate appropriate texts and save the file giving it appropriate name (check here for more). For example French: fr_FR, German: de_DE, etc.
The last step is setting the language in wp-config.php file. Please open it and find (or add if it doesn't exists) below line: 

define ('WPLANG', '');
Then set your language, for example French: 
define ('WPLANG', 'fr_FR');
That's all.
